# 🛒 Django Ecommerce Website

A simple **Ecommerce Web Application** built using Django.
Users can browse products, view product details, and manage products through the admin panel.

---

## 📌 Features

* User Registration and Login
* Product Listing Page
* Product Detail Page
* Add Products via Admin Panel
* Admin Dashboard
* Responsive UI

---

## 🛠 Technologies Used

* Python
* Django
* HTML
* CSS
* Bootstrap
* Mysql

---

## 🚀 Installation Guide

### 1️⃣ Clone the Repository

git clone https://github.com/student-vivekhirpara/django-ecommerce.git

### 2️⃣ Go to Project Folder

cd django-ecommerce

### 3️⃣ Install Dependencies

pip install -r requirements.txt

### 4️⃣ Run the Server

python manage.py runserver

### 5️⃣ Open in Browser

http://127.0.0.1:8000/
